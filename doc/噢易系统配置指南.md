# 噢易系统配置指南
## 管理平台基本使用
* 获取噢易系统的管理面板地址和登录用户名密码，进行登录，注意管理面板只能同时登录一个客户端，多人操作请注意
* 登录后请熟悉左侧导航栏的`桌面`、`模版`、`终端`的界面
### 制作新系统镜像
1. 进入`模版`-`教学模版`
2. 左上角选择对应机房，基础系统有`xxx教学`、`xxx考试`、`xxx其他考试`，点击对应模版的右下的三个点，后进入`配置属性`，先将`ipv4`和`ipv6`关闭，这步的目的是保证克隆镜像时地址池有足够的ip可以分配，否则会失败（可以遍历所有模版，关闭所有的地址分配）
3. 选择底板镜像，点击模版下方的克隆，从左向右第二个图标，命名后，选择全量复制，等待克隆完成
4. 点击模版下方的控制台，从左向右第一个图标，此时会在噢易的管理系统母机上开启虚拟机运行该镜像，确保开机运行正常
5. 根据新系统环境配置的复杂难易程度，较少操作可以直接在虚拟机上修改，完成修改后点击右上角更新镜像即可，如果较多修改且不方便在虚拟机操作，需要将本克隆镜像下发至实体机进入编辑模式进行操作，请见后文。
### 系统镜像（桌面）创建
1. 进入`桌面`-`教学桌面`
2. 左上角选择对应机房，后点击`新增`，填写场景名和机房，下方选择新建的镜像，点击下一步
3. 计算机名可以保持默认，以教学系统为例，他的前缀是stu，可以根据具体情况修改前缀名，但一般没必要；系统盘是否还原根据实际使用决定（等价于还原卡，只不过这个软还原，即系统层面决定是否还原，而非硬件还原卡）；场景数据盘如果开启了不还原系统就没必要勾选了，如要勾选，也可选择还原频率和大小；其余部分保持默认即可，一直下一步到完成即可。
4. 等待创建完成，打开激活开关即可
### 系统镜像（桌面）下发
1. 进入`终端`-`终端管理`
2. 选择左上机房，等待加载完成，此时需要同步将对应机房机器开机（使用对应机房的教师端噢易进行开机），进入任意系统即可
3. 选择你要下发系统的机器（可全选也可单选），点击操作拦`部署`-`下桌面`,选择需下发桌面，传输方式选择BT，下发即可
4. 此时对应机器的下发任务会显示进度，对应机器左下也会有下发进度，如果没有可以手动状态栏找到一个蓝色图标，右键数据传输也可以看到进度（但有些机房系统有bug，下发会导致状态栏图标崩溃，此时参考管理面板下发进度即可）
5. 等待下发完成，重启机器，选择下发的系统进行校验
6. 机器批量切换系统有两种方式
    1. 在对应机房教师端点击右上角`上课`,随后选择对应系统确认，然后点击导航栏远程开机即可
    2. 在管理端的`终端`-`终端管理`，选择对应机房的全部机器，点击操作拦的场景切换也可进行系统切换，但要注意需要所有机器先开机进任意系统，否则切换失败（猜测是交换机vlan隔离了）
### 对于较大体量的系统定制
1. 首先先将系统克隆好，确保克隆系统可以进入
2. 选择对应机房的一台机器下发该克隆镜像，传输完后检查是否能正常进入
3. 重启机器，待进入系统选择界面，用方向键或鼠标将光标切换至对应系统位置，键盘`ctrl + e`，输入管理员账户和密码，回车进入编辑模式，如果是不还原的桌面会有提示，清空编辑层确认即可
4. 进入系统后，确保获取到了正确的机器ip，会有弹窗提示进入编辑模式，或右键左下的蓝色应用图标，查看状态为编辑模式
5. 安装需要的环境，对系统进行配置
6. 配置完成后，重启，进入系统选择界面，用方向键或鼠标将光标切换至对应系统位置，键盘`ctrl + e`，推出编辑模式，随后回车进入系统
7. 检查系统是否正确安装了需要的环境，修改是否都生效
8. 确认无误后，左下的蓝色应用图标，点击更新模版，输入管理员账号密码，选择需要更新覆盖的镜像，这里选择系统正在运行的镜像（克隆的），注意不要覆盖错系统，检查更新状态，更新即可，此时会上传更新覆盖教学模版
9. 等待更新完成，管理面板的教学模版处模版由灰色锁定变为蓝色正常状态，再下发至所有设备即可
